# 🚀 PepOnline

**PepOnline** (Precision Engineering Portal) is a sleek, local-first daily log and task manager designed for developer productivity. 

## ✨ Key Features
- **Local-First Architecture:** Your tasks stay on your device, private and secure within the browser's storage.
- **Micro-Animations & Modern UI:** A premium experience powered by React, TailwindCSS, and sleek glassmorphism effects.
- **Categorized Daily Logs:** Automatically group and sort your logs by day to maintain a historical record of your progress.
- **Hyper-Fast Interface:** Built with Vite for nearly instantaneous load and interaction speeds.

## 🛠️ Tech Stack
- **React 19:** For building dynamic and responsive UI components.
- **TailwindCSS 4:** Using the latest styling techniques for a modern aesthetic.
- **Lucide Icons:** (Coming Soon) Integration with lightweight social icons.
- **Local Storage:** Persistent data management without backend complexity.
